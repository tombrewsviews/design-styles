# Design Styles Reference Skill

A comprehensive guide to **50 design aesthetics** for creating visually distinctive work. Each style includes detailed documentation of:

- **Core Principle** — The foundational philosophy driving the aesthetic
- **Visual Characteristics** — Specific visual elements, techniques, and approaches
- **Color Palette** — Typical colors, combinations, and color relationships
- **Typography** — Font styles, text treatments, and typographic approaches
- **Key Elements** — Motifs, patterns, recurring imagery, and signature elements
- **Feeling** — Emotional and conceptual impact the style creates
- **Era/Context** — Historical background, timeline, and cultural origins
- **Prompt Keywords** — AI image generation-optimized terms

## How to Use This Skill

### For Visual Design Work
1. Identify the mood, era, or feeling you want to evoke
2. Find matching styles in the index above
3. Read the detailed entry in the corresponding reference file
4. Apply visual characteristics, colors, typography, and key elements
5. Use prompt keywords for AI image generation if needed

### For AI Image Generation
Each style includes **prompt engineering keywords** optimized for diffusion models (Midjourney, DALL-E, Stable Diffusion). Combine:

```
[Subject] + [Style keywords from entry] + [Color palette terms] + [Additional modifiers]
```

**Example prompts:**

**Swiss Style (#1):**
```
corporate poster design, swiss style, helvetica typography, 
mathematical grid composition, red black white, clean minimal, 
müller-brockmann inspired
```

**Cyberpunk (#31):**
```
futuristic city street at night, cyberpunk aesthetic, neon signs, 
rain-slicked pavement, blade runner style, holographic advertisements, 
neon pink cyan blue, corporate dystopia
```

**Vaporwave (#30):**
```
greek marble bust, vaporwave aesthetic, pink purple gradient background, 
windows 95 elements, japanese text, palm trees, VHS glitch effects, 
mall aesthetic, sunset colors
```

**Punk (#14):**
```
concert poster, punk aesthetic, ransom note cut-out typography, 
jamie reid style, xerox texture, safety pins, anarchist symbols, 
black and white, raw aggressive DIY
```

**Memphis Design (#41):**

## Style Combinations

Many compelling designs blend multiple aesthetics:

| Combination | Result |
|-------------|--------|
| Cyberpunk + Art Deco | Blade Runner's visual language |
| Vaporwave + Brutalism | Digital decay meets concrete monumentality |
| Gothic + Steampunk | Victorian dark romance with mechanical elements |
| Swiss Style + Neon colors | Clean grids with electric energy |
| Neubrutalism + Y2K | Raw digital energy meets millennium optimism |
| Punk + Glitch | Analog rebellion meets digital corruption |
| Minimalism + Gothic | Dark elegance with maximum restraint |
| 70s Retro + Synthwave | Double-layered nostalgia |
| Memphis + Cyberpop | 80s postmodern meets 90s neon optimism |
